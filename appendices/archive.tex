\chapter{Thesis archive contents}\label{appx:source}

The digital thesis archive\footnote{Available at \url{https://is.muni.cz/th/ddfps/}.} contains the following items:

\begin{itemize}
    \item \texttt{thesis.pdf}: the text of this thesis as a PDF document,
    \item \texttt{svglab.zip}: the complete source code of the \textsc{svglab} library.
\end{itemize}

The ZIP archive with the source code contains, among others, the following:
\begin{itemize}
    \item \texttt{svglab}: the \textsc{svglab} Python package,
    \item \texttt{tests}: the test suite,
    \item \texttt{LICENSE}: the text of the MIT license,
    \item \texttt{README.md}: installation and usage instructions.
\end{itemize}
